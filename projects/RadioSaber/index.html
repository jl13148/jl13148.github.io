RadioSaber Extension

Computer Networking Final Project

Final Project — CSCI-SHU 308 (Computer Networking), Fall 2023
New York University Shanghai
Advisor: Prof. Guyue Liu
Github Repository: RadioSaber Extension
Team Members: Yuncheng (Peter) Yao, Jinting Liu, Charlie Xiao

Overview

Motivated by our analysis of RadioSaber (Chen et al., 2023), we identified a tension between inter-slice throughput optimization and diverse intra-slice performance objectives. To address this mismatch, we designed RadioSaber Extension, which switches the resource allocation strategies across time. This approach allows intra-slice goals to be better satisfied while incurring only a moderate sacrifice in overall throughput.
